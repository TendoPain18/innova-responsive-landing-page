## 🎨 About The Project
Innova is a financial literacy platform designed to equip future generations with essential money management skills. The landing page presents the platform's mission, value proposition, partner school logos, testimonials from users, a summer program section, and a collaboration contact form — all built with semantic HTML and custom CSS using a teal and lime-green color palette.

## ✨ Features

- **Responsive Multilingual Navbar**: Fixed navigation with logo, desktop nav links, login/demo buttons, a language switcher dropdown (Arabic, English, French with flag icons), and a mobile hamburger context menu
- **Animated Landing Section**: Full-height hero with animated money paper background effect, a diagonal bomb decoration animation, and a "Join Now" CTA button
- **Problem Section**: Split layout with descriptive text and an illustration, highlighting the platform's core mission
- **Why Innova Section**: Fully responsive image-swap layout — different images served for mobile, tablet, and desktop screen sizes
- **Swiper Testimonials Slider**: Centered auto-playing testimonial cards with quote icons, star ratings, numbered pagination bullets, and per-slide video support
- **Partner School Logos**: Five school logo badges displayed below the testimonials slider
- **Summer Program Section**: Split layout with an illustration and a checklist of program highlights
- **Contact / Collaborate Form**: Full contact form with name, email, phone, and message fields with custom validation messages
- **Dark Footer**: Logo, tagline, social media icon links (Facebook, Instagram, LinkedIn, TikTok, WhatsApp), and copyright
- **Fixed Scroll-to-Top Button**: Animated floating button

## 🚀 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/TendoPain18/innova-responsive-landing-page.git
```

2. **Open in browser**
```
Open index.html directly in any modern browser
```

No build tools or dependencies required.

## 🛠️ Built With

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Flexbox, CSS Grid, keyframe animations, media queries
- **JavaScript** — Custom context menu toggle and Swiper initialization
- **Swiper.js 5** — Touch slider for testimonials with auto-play and numbered pagination
- **Font Awesome 6** — Icons
- **Google Fonts** — Cairo typeface
